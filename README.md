# Volleyball Player Base

## Description
[Write what your app does here — for example:]
A web application built with Django that allows users to manage a volleyball player database.
Users can view, add, update, and delete players from the list.

## Features
- View all players
- Add a new player
- Update an existing player
- Delete a player

## Technologies Used
- Python
- Django
- SQLite
- Docker

## How to Run with Docker

### Pull and run the image from GitHub:
```bash
docker pull ghcr.io/fadel-saleh/eece430sp25-26fadel_saleh-volley-list:latest
docker run -p 8000:8000 ghcr.io/fadel-saleh/eece430sp25-26fadel_saleh-volley-list:latest
```
Then open your browser and go to: http://localhost:8000
